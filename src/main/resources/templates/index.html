Le projet

Ce portail client est une application web développée en Java avec le framework Spring Boot. Il permet aux utilisateurs de gérer des informations sur les clients, y compris leur nom, leur email, leur adresse et les services qu'ils ont reçus. Les utilisateurs peuvent ajouter, modifier et supprimer des clients. Cette application utilise Thymeleaf pour les templates HTML et est sécurisée avec OAuth2.
